 🏋️ Gym Website

A modern and responsive Gym Management Website built using **HTML**, **CSS**, **JavaScript**, **Node.js**, and **MySQL**. The project provides an interactive platform where users can explore gym services, register, log in, and manage their membership information through a secure backend connected to a MySQL database.

 Features

- Responsive and user-friendly interface
- User Registration and Login
- Secure authentication using Node.js
- Membership management
- Contact form for customer inquiries
- Dynamic content powered by JavaScript
- Backend API built with Node.js
- MySQL database integration for storing user and membership data
- Clean and organized project structure
